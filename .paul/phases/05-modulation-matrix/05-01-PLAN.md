<task type="auto">
  <name>Task 2: Implement Lfo class</name>
  <files>Source/Modulation/Lfo.h, Source/Modulation/Lfo.cpp</files>
  <action>
    Create Source/Modulation/Lfo.h:

    ```cpp
    #pragma once
    #include <juce_audio_basics/juce_audio_basics.h>
    #include <array>
    #include <atomic>

    class Lfo
    {
    public:
        static constexpr int kDrawablePoints = 64;

        enum class Waveform { Sine = 0, Triangle, Saw, Square, Random, SAndH, Drawable };

        Lfo() = default;

        void prepare(double sampleRate) noexcept;
        void reset() noexcept;

        void setParamPointers(std::atomic<float>* rate,
                              std::atomic<float>* waveform,
                              std::atomic<float>* sync,
                              std::atomic<float>* phase,
                              std::atomic<float>* depth) noexcept;

        // Audio-rate: advance phase by one sample, return scaled value [-depth, +depth].
        // Called from audio thread.
        float advanceSample() noexcept;

        // Control-rate: return last computed value without advancing.
        float currentValue() const noexcept { return lastValue_; }

        // Block-level call (audio thread): update sync interval from bpm.
        // Pass numSamples for sync phase computation.
        void processBlock(int numSamples, double bpm) noexcept;

        // UI thread: edit drawable waveform (index 0..63, value -1..+1).
        void setDrawablePoint(int index, float value) noexcept;
        float getDrawablePoint(int index) const noexcept;

        JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR(Lfo)

    private:
        float computeSample(float normPhase, Waveform wf) noexcept;
        float sampleDrawable(float normPhase) const noexcept;

        double phase_{ 0.0 };          // [0..1)
        double sampleRate_{ 44100.0 };
        float  lastValue_{ 0.0f };

        // S&H state
        float  sAndHHeld_{ 0.0f };
        float  prevPhase_{ 0.0f };

        // Random waveform state (resample each cycle)
        float  randomCurrent_{ 0.0f };
        float  randomNext_{ 0.0f };
        float  prevPhaseRnd_{ 0.0f };

        // Drawable waveform — 64 atomic floats, edited from UI, read from audio
        std::array<std::atomic<float>, kDrawablePoints> drawable_{};

        juce::Random rng_;

        std::atomic<float>* pRate_     { nullptr };
        std::atomic<float>* pWaveform_ { nullptr };
        std::atomic<float>* pSync_     { nullptr };
        std::atomic<float>* pPhase_    { nullptr };
        std::atomic<float>* pDepth_    { nullptr };
    };
    ```

    Create Source/Modulation/Lfo.cpp:

    - prepare(): set sampleRate_, reset()
    - reset(): phase_ = 0, lastValue_ = 0, clear S&H and random state; fill drawable_ with 0.0f
    - setParamPointers(): assign all pointers
    - setDrawablePoint(i, v): drawable_[clamp(i,0,63)].store(clamp(v,-1,1), relaxed)
    - getDrawablePoint(i): drawable_[clamp(i,0,63)].load(relaxed)

    - advanceSample():
      1. Read params (relaxed): rate (Hz), waveform index, sync bool, phaseOffset (0-360 → 0-1), depth
      2. Determine phase increment:
         - Free: increment = rate / sampleRate_
         - Sync: increment computed by processBlock(); no per-sample rate computation needed.
           For sync mode, use the stored syncIncrement_ (set in processBlock).
      3. Apply phase offset: effectivePhase = fmod(phase_ + phaseOffset/360.0, 1.0)
      4. Compute sample = computeSample(effectivePhase, wf)
      5. Scale by depth: lastValue_ = sample * depth
      6. Advance: phase_ = fmod(phase_ + increment, 1.0)
      7. Return lastValue_

    - computeSample(normPhase, wf):
      - Sine:     sin(normPhase * 2π)
      - Triangle: 1 - abs(normPhase * 4 - 2) - 1   (i.e., 2*abs(2*p - 1) - 1 gives [-1,+1])
                  Actually: tri(p) = 1.0 - 4.0*abs(p - 0.5)  gives [+1 at p=0, -1 at p=0.5, +1 at p=1]
                  Better: tri(p) = p < 0.5 ? (4.0f*p - 1.0f) : (3.0f - 4.0f*p)
      - Saw:      2*normPhase - 1  (rises from -1 to +1 over cycle)
      - Square:   normPhase < 0.5 ? +1.0f : -1.0f
      - Random:   smooth random (interpolated between randomCurrent_ and randomNext_).
                  On each cycle start (prevPhaseRnd_ > normPhase crossing), shift: randomCurrent_=randomNext_,
                  randomNext_=rng_.nextFloat()*2-1.
                  Return lerp(randomCurrent_, randomNext_, normPhase)
      - S&H:      on phase reset (prevPhase_ > normPhase crossing), sAndHHeld_ = rng_.nextFloat()*2-1.
                  Return sAndHHeld_
      - Drawable: call sampleDrawable(normPhase)

    - sampleDrawable(normPhase):
      float idx = normPhase * kDrawablePoints;
      int i0 = (int)idx % kDrawablePoints;
      int i1 = (i0 + 1) % kDrawablePoints;
      float frac = idx - floor(idx);
      float v0 = drawable_[i0].load(relaxed);
      float v1 = drawable_[i1].load(relaxed);
      return v0 + frac * (v1 - v0);

    - processBlock(numSamples, bpm):
      if sync enabled and bpm > 0: syncIncrement_ = bpm / 60.0 / sampleRate_ * numSamples
      (sync LFO advances phase by numSamples per block based on bpm — simplified for now,
       full per-sample sync stays free-running with tempo-matched rate)
      Actually for sync mode: just compute rate_hz = bpm / 60.0 (one cycle per beat = quarter note sync)
      and let advanceSample use that rate. Store as syncRateHz_. advanceSample uses syncRateHz_
      when sync=true instead of pRate_->load().

    Avoid:
    - Any heap allocation in advanceSample (no std::vector, no new)
    - std::mutex or any lock in audio-thread path
    - Calling prevPhaseRnd_/prevPhase_ reset more than once per crossing
