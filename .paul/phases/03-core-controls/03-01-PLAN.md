<objective>

## Goal

Wire all 8 Grano parameters (`grainSize`, `density`, `position`, `positionJitter`, `pitchShift`, `stereoSpread`, `masterVolume`, `loop`) through `juce::AudioProcessorValueTreeState` (APVTS), have `GranularEngine::scheduleGrain()` read them at grain generation time, and implement preset save/load.

## Purpose

APVTS is the single source of truth for all parameters per ADR-003. Every subsequent phase (controls UI in 03-02, LFOs in F5, automation in F7) depends on these parameter IDs and their ranges existing and being correct. Without this plan, the engine still runs with hardcoded constants.

## Output

- `Source/Parameters.{h,cpp}` — parameter IDs + `createParameterLayout()`
- `PluginProcessor` — APVTS member, `getStateInformation/setStateInformation`, `setParamPointers()` call
- `GranularEngine` — reads 8 param atomics in `scheduleGrain()` and `processBlock()`; stochastic jitter via `juce::Random`
- `Tests/test_parameters.cpp` — 4 Catch2 tests (≥ 36 total)

</objective>

    **GranularEngine.cpp** — replace `scheduleGrain()` body with param-reading version:
    ```cpp
    void GranularEngine::scheduleGrain() noexcept
    {
        using ro = std::memory_order_relaxed;

        const float* srcData = nullptr;
        int          srcLen  = 0;

        if (sampleSource_ != nullptr)
        {
            srcData = sampleSource_->getReadPointer();
            srcLen  = sampleSource_->getNumSamples();
        }

        if (srcData == nullptr || srcLen <= 0)
        {
            if (!sampleReady_.load(std::memory_order_acquire))
                return;
            srcData = testSample_.data();
            srcLen  = static_cast<int>(sampleRate_);
        }

        // Read params with fallbacks for null pointers (headless test context).
        const float grainSizeMs = pGrainSize_      ? pGrainSize_->load(ro)      : 100.0f;
        const float position    = pPosition_       ? pPosition_->load(ro)       : 0.0f;
        const float jitter      = pPositionJitter_ ? pPositionJitter_->load(ro) : 0.0f;
        const float pitchShift  = pPitchShift_     ? pPitchShift_->load(ro)     : 0.0f;
        const float spread      = pStereoSpread_   ? pStereoSpread_->load(ro)   : 0.5f;
        const bool  doLoop      = pLoop_           ? pLoop_->load(ro) >= 0.5f   : false;

        Grain* g = pool_.acquire();
        if (g == nullptr)
            return;

        const int grainLen = std::max(1, static_cast<int>(grainSizeMs * 0.001f * sampleRate_));

        // Compute start position
        int startPos = static_cast<int>(position * static_cast<float>(srcLen));
        if (jitter > 0.0f && srcLen > 1)
        {
            const int maxJitter = static_cast<int>(jitter * 0.5f * static_cast<float>(srcLen));
            if (maxJitter > 0)
                startPos += grainRng_.nextInt(2 * maxJitter + 1) - maxJitter;
        }
        if (doLoop)
            startPos = ((startPos % srcLen) + srcLen) % srcLen;
        else
            startPos = std::clamp(startPos, 0, std::max(0, srcLen - 1));

        const int available = srcLen - startPos;
        if (available <= 0) { pool_.release(g); return; }

        g->source        = srcData;
        g->startPos      = startPos;
        g->lengthSamples = std::min(grainLen, available);
        g->pitchRatio    = std::pow(2.0f, pitchShift / 12.0f);
        g->pan           = spread > 0.0f ? (grainRng_.nextFloat() * 2.0f - 1.0f) * spread : 0.0f;
        g->shape         = EnvelopeShape::Hann;
        g->currentPhase  = 0.0f;

        int s1, n1, s2, n2;
        fifo_.prepareToWrite(1, s1, n1, s2, n2);
        if (n1 > 0)       { fifoSlots_[s1] = g; fifo_.finishedWrite(1); }
        else if (n2 > 0)  { fifoSlots_[s2] = g; fifo_.finishedWrite(1); }
        else              { pool_.release(g); }
    }
    ```
