  <action>
    **SampleBuffer** manages a hot-swappable audio buffer with three-state atomics:
    - `pending_`: loading thread writes here; audio thread reads + claims
    - `current_`: audio thread owns exclusively (the live read source)
    - `retired_`: audio thread deposits old buffer here; message thread deletes

    **SampleBuffer.h:**
    ```cpp
    #pragma once
    #include <juce_audio_basics/juce_audio_basics.h>
    #include <atomic>
    #include <memory>

    // RT-safe hot-swappable sample buffer.
    //
    // Threading contract:
    //   setPending()      — message thread / loading thread only
    //   trySwap()         — audio thread only (call once per processBlock)
    //   processRetired()  — message thread only (call from Timer at ~30 Hz)
    //   getReadPointer()  — audio thread only, after trySwap()
    //   getNumSamples()   — audio thread only, after trySwap()
    class SampleBuffer
    {
    public:
        SampleBuffer() = default;
        ~SampleBuffer();

        // Loading thread: post a new buffer. Ownership is transferred.
        // If a prior pending buffer was never claimed, it is leaked until
        // processRetired() is called — but only one buffer can be pending at once,
        // so this replaces any unclaimed pending.
        void setPending(std::unique_ptr<juce::AudioBuffer<float>> newBuf);

        // Audio thread: atomically claim pending buffer, retire old current.
        // Returns true if a new buffer was claimed. noexcept, no allocation.
        bool trySwap() noexcept;

        // Audio thread: read pointer to current channel 0 data. nullptr if empty.
        const float* getReadPointer() const noexcept;

        // Audio thread: number of samples in current buffer. 0 if empty.
        int getNumSamples() const noexcept;

        // Message thread (Timer callback): delete any retired buffer.
        void processRetired();

    private:
        // owned by audio thread
        std::unique_ptr<juce::AudioBuffer<float>> current_;

        // raw pointers for atomic operations (ownership transferred via pointer)
        std::atomic<juce::AudioBuffer<float>*> pending_{ nullptr };
        std::atomic<juce::AudioBuffer<float>*> retired_{ nullptr };

        JUCE_DECLARE_NON_COPYABLE_WITH_LEAK_DETECTOR(SampleBuffer)
    };
    ```

    **SampleBuffer.cpp:**

    1. **setPending()**: `delete pending_.exchange(newBuf.release(), std::memory_order_acq_rel)` — if there was already an unclaimed pending buffer, delete it (only happens if two loads fire before one processBlock; rare but must not leak).

    2. **trySwap()** — audio thread, noexcept, no allocation:
       ```cpp
       auto* p = pending_.load(std::memory_order_acquire);
       if (p == nullptr) return false;
       if (!pending_.compare_exchange_strong(p, nullptr,
               std::memory_order_acq_rel, std::memory_order_relaxed))
           return false;
       // Retire old current (message thread will delete)
       auto* old = current_.release();
       if (old != nullptr)
           retired_.store(old, std::memory_order_release);
       current_.reset(p);
       return true;
       ```

    3. **getReadPointer()**: `return current_ ? current_->getReadPointer(0) : nullptr;`

    4. **getNumSamples()**: `return current_ ? current_->getNumSamples() : 0;`

    5. **processRetired()**: 
       ```cpp
       auto* r = retired_.exchange(nullptr, std::memory_order_acq_rel);
       delete r;
       ```

    6. **~SampleBuffer()**:
       - `delete pending_.load()` (if loading thread left something behind after plugin destruction)
       - `delete retired_.load()`
       - `current_` unique_ptr destructor handles current

    **Do NOT**: allocate in trySwap(), use std::mutex anywhere, call processRetired() from the audio thread.

      ```cpp
      bool GranoAudioProcessor::loadSampleFile(const juce::File& file)
      {
          std::unique_ptr<juce::AudioFormatReader> reader(
              formatManager_.createReaderFor(file));
          if (reader == nullptr) return false;

          const int numSamples = (int)reader->lengthInSamples;
          if (numSamples <= 0) return false;

          auto buf = std::make_unique<juce::AudioBuffer<float>>(
              (int)reader->numChannels, numSamples + 2); // +2 guard samples
          buf->clear();

          if (!reader->read(buf.get(), 0, numSamples, 0, true, true))
              return false;

          // Zero the guard samples to prevent interpolation OOB reads
          for (int ch = 0; ch < buf->getNumChannels(); ++ch)
          {
              buf->setSample(ch, numSamples,     0.0f);
              buf->setSample(ch, numSamples + 1, 0.0f);
          }

          sampleBuffer_.setPending(std::move(buf));
          engine_.notifyNewSource(); // signals engine to pick up on next processBlock
          return true;
      }
      ```
