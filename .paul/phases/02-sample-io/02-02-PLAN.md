<objective>

## Goal

Build the `WaveformDisplay` UI component: renders the loaded sample via `juce::AudioThumbnail`, overlays active grain particles (`#C77CFF`, opacity ∝ envelope amplitude), draws a playhead cursor (`#7CF5C4`), and shows file/duration/grain-count labels — all per `DESIGN_SPEC.md`. Wire it into `PluginEditor` as the centerpiece. Add a drag-highlight overlay for the drop-target affordance.

## Purpose

Completes F2. After this plan, the user sees a visual representation of their loaded sample and live grain activity. Every subsequent phase builds controls that affect this display. Without it, the plugin is visually inert.

    ```cpp
    struct GrainSnapshot
    {
        float srcFraction;  // grain start position as [0..1] of loaded sample length
        float envelopeAmp;  // current envelope amplitude [0..1] for particle opacity
    };

    // UI thread (30 Hz Timer): copy current active grain snapshot data.
    // Returns the number of grains written into `out` (≤ maxCount).
    // Note: technically racy on the snapshot array (audio thread writes, UI reads)
    // but aligned-float writes are atomic on x86 and visual tearing is acceptable.
    // TSAN does not exercise this path in CI (no MessageManager in tests).
    int getGrainSnapshots(GrainSnapshot* out, int maxCount) const noexcept;
    ```

    Add private members:
    ```cpp
    std::array<GrainSnapshot, MaxActiveGrains> grainSnapshots_{};
    std::atomic<int>                           grainSnapshotCount_{ 0 };
    ```

    **GranularEngine.cpp** — at the END of `processBlock()`, after grain rendering loop,
    before returning:
    ```cpp
    // Write snapshot for UI particle rendering (called by getGrainSnapshots on UI thread).
    const int snapLen = sampleSource_ != nullptr ? sampleSource_->getNumSamples() : 0;
    const int snapCount = std::min(activeGrainCount_, MaxActiveGrains);
    for (int i = 0; i < snapCount; ++i)
    {
        const auto* g = activeGrains_[i];
        const float srcFrac = snapLen > 0
            ? std::clamp((float)g->startPos / (float)snapLen, 0.0f, 1.0f)
            : 0.0f;
        grainSnapshots_[static_cast<std::size_t>(i)] = {
            srcFrac,
            applyEnvelope(g->shape, g->currentPhase)
        };
    }
    grainSnapshotCount_.store(snapCount, std::memory_order_release);
    ```

    Add definition:
    ```cpp
    int GranularEngine::getGrainSnapshots(GrainSnapshot* out, int maxCount) const noexcept
    {
        const int count = std::min(grainSnapshotCount_.load(std::memory_order_acquire), maxCount);
        for (int i = 0; i < count; ++i)
            out[i] = grainSnapshots_[static_cast<std::size_t>(i)];
        return count;
    }
    ```

    **Tests/test_engine.cpp** — add one test after the existing threading test:
    ```cpp
    TEST_CASE("GranularEngine getGrainSnapshots returns plausible data", "[GranularEngine]")
    {
        GranularEngine engine;
        engine.prepare(48000.0, 512);
        juce::Thread::sleep(100); // let scheduler fill grains from testSample_

        juce::AudioBuffer<float> out(2, 512);
        engine.processBlock(out); // writes snapshots

        std::array<GranularEngine::GrainSnapshot, GranularEngine::MaxActiveGrains> snaps;
        const int count = engine.getGrainSnapshots(snaps.data(), GranularEngine::MaxActiveGrains);

        REQUIRE(count >= 0);
        REQUIRE(count <= GranularEngine::MaxActiveGrains);
        for (int i = 0; i < count; ++i)
        {
            REQUIRE(snaps[i].srcFraction >= 0.0f);
            REQUIRE(snaps[i].srcFraction <= 1.0f);
            REQUIRE(snaps[i].envelopeAmp >= 0.0f);
            REQUIRE(snaps[i].envelopeAmp <= 1.0f);
        }
        engine.reset();
    }
    ```

    ```cpp
        // Message thread: called after a sample is loaded successfully.
    ```
